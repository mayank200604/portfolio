## 💻 Running Locally

To view the portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mayank200604/portfolio.git
    ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
    ```
3. Open index.html in any web browser.


📦 Deployment

The site is deployed using GitHub Pages.

Deployment steps:

Push changes to the main branch

GitHub Pages automatically serves the site from the repository

Updates go live at:
https://mayank200604.github.io/portfolio/
